## 🌍 Hosting (GitHub Pages)

This site is hosted via **GitHub Pages** from the `main` branch root.

To host your own fork:
1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://<your-username>.github.io/HMG_Portfolio`
